Hệ Thống MiniCloud Mô Phỏng Cloud Platform

Một dự án mô phỏng kiến trúc AWS / GCP / Azure với 9 loại server.

Khám phá Blog

Các thành phần chính

🌐 Web Frontend

Trang web tĩnh chạy trên Nginx, cung cấp giao diện người dùng.

⚙️ Backend API

Microservice Flask xử lý nghiệp vụ và kết nối đến các server khác.

💾 Database (MariaDB)

Lưu trữ dữ liệu người dùng, metadata và thông tin hệ thống.

🔐 Authentication (Keycloak)

Cung cấp đăng nhập, cấp token và quản lý người dùng.

📣 Notification Server

Xử lý gửi thông báo email hoặc sự kiện hệ thống.

📦 Storage Server (MinIO)

Lưu trữ file, object storage tương tự AWS S3.

📝 Logging Server

Thu thập và lưu lại log từ mọi dịch vụ để dễ theo dõi.

⚡ Cache Server (Redis)

Lưu cache tạm thời giúp tăng tốc độ truy vấn.

📨 Message Queue (RabbitMQ)

Hỗ trợ giao tiếp bất đồng bộ giữa các microservice.
