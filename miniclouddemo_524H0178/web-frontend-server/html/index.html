9 Thành Phần Hệ Thống

Mỗi thành phần chạy trong container riêng biệt, giao tiếp qua API và message queue, phản ánh đúng kiến trúc cloud thực tế.

🌐

Web Frontend

Máy chủ Nginx phục vụ giao diện tĩnh (HTML/CSS). Điểm vào duy nhất từ phía người dùng, cấu hình reverse proxy linh hoạt.

Nginx · Static
⚙️

Backend API

REST API xử lý toàn bộ nghiệp vụ hệ thống. Nhận request từ frontend, điều phối các dịch vụ nội bộ và trả kết quả.

Flask · REST API
🗄️

Database (MariaDB)

Hệ quản trị cơ sở dữ liệu quan hệ lưu trữ toàn bộ dữ liệu nghiệp vụ, người dùng và cấu hình hệ thống.

MariaDB · RDBMS
🔐

Authentication (Keycloak)

Giải pháp xác thực và phân quyền (Identity Provider). Hỗ trợ OAuth 2.0, OpenID Connect và Single Sign-On.

Keycloak · OAuth2 · SSO
🔔

Notification Server

Dịch vụ gửi thông báo đa kênh: Email, SMS. Nhận sự kiện từ hàng đợi và xử lý phân phối thông điệp.

Email · SMS · Event
📦

Storage Server (MinIO)

Lưu trữ đối tượng (Object Storage) tương thích S3. Quản lý file upload, media và backup dữ liệu nhị phân.

MinIO · S3-Compatible
📊

Logging Server

Hệ thống thu thập và phân tích log tập trung. Theo dõi toàn bộ hoạt động và giám sát hiệu năng hệ thống.

Logging · Monitoring
⚡

Cache Server (Redis)

In-memory data store tốc độ cao dùng cho cache, session và rate limiting. Giảm tải đáng kể cho database chính.

Redis · Cache · Session
🐇

Message Queue (RabbitMQ)

Message broker điều phối giao tiếp bất đồng bộ giữa các service. Đảm bảo độ tin cậy và khả năng mở rộng.

RabbitMQ · AMQP · Async
